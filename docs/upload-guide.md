# Upload Guide

このファイルは、解答コードを継続的に追加するための運用メモです。

## 1. ブランチ方針
- まずは main 直コミットで OK（小規模・個人運用のため）
- 慣れてきたら feature ブランチ運用に移行

## 2. 1 回の追加単位
- 最小単位: 1 問 1 コミット
- まとめたい場合でも 3 問程度まで

## 3. 推奨コマンド

git add .
git commit -m "apg4b: add q001"
git push origin main

## 4. 詰まった時の記録
- 10 分以上詰まったら、いったん方針をコメントに残す
- 解けたら「何が詰まりポイントだったか」を 1 行で追記

## 5. 継続しやすくする工夫
- 毎日 1 問 or 週 5 問のどちらかを目標に固定
- 問題を解く時間帯を固定（例: 平日 22:00-22:30）
- 完璧より継続を優先
